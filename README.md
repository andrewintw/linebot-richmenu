
```bash
$ ./richmenu_ctrl.sh
Usage:
	richmenu_ctrl.sh list                         - 列出所有 Rich Menu
	richmenu_ctrl.sh create <menu.json>           - 根據 JSON 檔案建立 Rich Menu
	richmenu_ctrl.sh info <richMenuId>            - 查看特定 Rich Menu 的詳細資訊
	richmenu_ctrl.sh validate <menu.json>         - 驗證 Rich Menu 設定檔
	richmenu_ctrl.sh upload-image <richMenuId> <image_path> - 上傳 Rich Menu 圖片
	richmenu_ctrl.sh default <richMenuId>         - 設定預設 Rich Menu
	richmenu_ctrl.sh check-image <richMenuId>     - 檢查 Rich Menu 是否有圖片
	richmenu_ctrl.sh check-default                - 檢查目前的預設 Rich Menu
	richmenu_ctrl.sh delete <richMenuId>          - 刪除指定的 Rich Menu
	richmenu_ctrl.sh clear-default                - 清除預設的 Rich Menu

```
